# Rule-Based AI Chatbot

## Overview

Rule-Based AI Chatbot is a Python-based chatbot that interacts with users through predefined responses. The chatbot greets users based on the current time of the day and can also speak its responses using Text-to-Speech technology.

## Features

* Time-based greetings (Morning, Afternoon, Evening)
* User name recognition
* Text-to-Speech functionality using pyttsx3
* Motivational responses
* Study and programming guidance
* Rule-based conversation system
* Exit command using "bye"

## Technologies Used

* Python
* datetime module
* pyttsx3 library

## How to Run

1. Install Python on your system.
2. Install pyttsx3 library:

```bash
pip install pyttsx3
```

3. Run the program:

```bash
python rule_based_ai_chatbot.py
```
